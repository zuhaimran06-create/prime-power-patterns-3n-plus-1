# Prime Power Patterns in Odd Numbers under 3n + 1 Transformation

This project explores how the prime factorization of an odd number changes under the transformation f(n) = 3n + 1. The goal is to analyze patterns in prime power decomposition and investigate whether these transformations follow predictable structures.

## Overview

- Analyze how prime factors and their exponents change after applying 3n + 1
- Study the introduction of new prime factors
- Explore whether patterns can be predicted

## Approach

- Generate large datasets of odd numbers
- Compute prime factorizations using computational tools
- Apply the transformation f(n) = 3n + 1
- Analyze changes in prime structure
- Explore machine learning approaches to predict transformations

## Concepts Covered

- Number theory (prime factorization)
- Computational mathematics
- Feature engineering
- Machine learning (Random Forests, Neural Networks)
- Pattern recognition in mathematical structures

## Notes

This project combines theoretical mathematics with computational experimentation to explore patterns in number transformations.
